
# EOQ + ABC Inventory Control Project

This Excel-based project analyzes inventory using:
- **EOQ (Economic Order Quantity)** to minimize total cost
- **ABC Classification** to prioritize items by value

## Sheets Included
- **Raw Data**: Item details, demand, cost
- **EOQ Analysis**: Optimal order size and cost breakdown
- **ABC Classification**: Value-based categorization with color-coded formatting
